# Tianquan — 天权·隐光知识系统

本仓库包含两个核心组件：

## 知识引擎（knowledge-engine）

多语言书籍索引与知识管理工具。

- **Tantivy 全文索引** — 支持中/日/韩/希伯来/英五种语言的分词与搜索
- **Kùzu 知识图谱** — 书籍元信息、分类关系、概念图谱
- **理解模块** — `understanding.db` 三层隔离架构（对话记忆/书籍索引/读书理解）
- **书籍导入** — 从 Markdown 源文件批量导入，保留原文 (`body_raw`) 和分词版本 (`body`)

## TrustForge（trustforge）

零信任隔离铸造机 — 防大模型走捷径的强制验证框架。

- **开发验证** — `parse` → `verify` → `compile` → `run`，强制 DSL 逻辑验证 + 沙盒执行
- **读书理解** — `store` → `verify` → `audit` → `derive`，三要素强制约束 + 证据验证 + 闭卷推导
- **符号推导** — 基于已存公理的推导引擎（行列式、线性代数、运动学），不依赖大模型参数
- **防幻觉架构** — Layer 1 原文提取 / Layer 2 evidence 验证 / Layer 3 闭卷审计

## 设计原则

> **理解不是我说我知道了，而是我能基于它做出正确的行动。**

- 所有输出必须附带可验证的证据
- 所有功能经过真实执行测试而非 AI 声明
- 开发数据与 AI Agent 个人数据物理隔离
